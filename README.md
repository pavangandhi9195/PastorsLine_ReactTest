# PastorsLine_ReactTest
Pastors Line - React developer test

If you want to run this test in mac machine please change the below script command in package.json file.
"scripts": 
{
    "start": "set PORT=30080 && react-scripts start", //For Mac use this in script : "PORT=30080 react-scripts start"
    "build": "react-scripts build",
    "test": "react-scripts test",
    "eject": "react-scripts eject"
},
